PROJECT DESCRIPTION

About project

We have designed a Library portal which can be used to manage digital copies as well as hard copies of various generes and publications. Users are of two types: 1.Admin 2.User, Admin can add copies, can make other people as admin and have all other privelages of normal user. Users can view/ download digital copies if avaiable.
Notifications for over-due is sent to all users through mail service.
A user can request a book for issue and when admins accepts user request for book , notification to student is sent to his portal.
Open resources are the one accesible to all


Requirements to run

  This project was done in April,2019. Requirements are flask,jinja2, python3, sqlalchemy or MySQL, sqlite3, HTML, CSS and javascript.
